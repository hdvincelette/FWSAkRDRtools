# Package index

  : Clone a project from the USFWS Alaska Regional Data Repository (RDR)
- [`commit.files()`](https://hdvincelette.github.io/FWSAkRDRtools/reference/commit.files.md)

  : Find the location(s) of project file(s) on the USFWS Alaska Regional
  Data Repository (RDR)
- [`find.projects()`](https://hdvincelette.github.io/FWSAkRDRtools/reference/find.projects.md)
  : Find projects on the USFWS Alaska Regional Data Repository (RDR)

  : Get contacts from an mdEditor metadata file
- [`get.dir.template()`](https://hdvincelette.github.io/FWSAkRDRtools/reference/get.dir.template.md)
  : Get the USFWS Alaska Regional Data Repository (RDR) project
  directory tree template.
- [`read.tables()`](https://hdvincelette.github.io/FWSAkRDRtools/reference/read.tables.md)
  : Read a tabular project file on the USFWS Alaska Regional Data
  Repository (RDR) into R
- [`scrape.mdEditor()`](https://hdvincelette.github.io/FWSAkRDRtools/reference/scrape.mdEditor.md)
  : Scrape metadata from an mdEditor metadata file

  : Summarize project files on the USFWS Alaska Regional Data Repository
  (RDR)
