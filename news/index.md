# Changelog

## FWSAkRDRtools 0.2.0

#### **Added**

- [`find.projects()`](https://hdvincelette.github.io/FWSAkRDRtools/reference/find.projects.md):
  find projects on the USFWS Alaska Regional Data Repository (RDR)

- Arguments to review local duplicate files and overwrite RDR files in
  [`commit.files()`](https://hdvincelette.github.io/FWSAkRDRtools/reference/commit.files.md)

- Arguments to specify header and na strings in
  [`read.tables()`](https://hdvincelette.github.io/FWSAkRDRtools/reference/read.tables.md)

- [FWSAkRDRtools
  Tutorial](https://hdvincelette.github.io/FWSAkRDRtools/articles/01_FWSAkRDRtools_Tutorial.html)
  article

&
