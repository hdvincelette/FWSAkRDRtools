Workspace

Uninstall

Run the following to uninstall FWSAkRDRtools.

remove.packages("FWSAkRDRtools")
.rs.restartR()

Install

Run the following to install the development version of FWSAkRDRtools. If you already have an older version of FWSAkRDRtools installed in your library, run the code above to uninstall before proceeding.

if (!require("remotes")) install.packages("remotes")

Sys.setenv(R_REMOTES_STANDALONE="true")
remotes::install_github("hdvincelette/FWSAkRDRtools")

Load

Run setup code.

options(width = 9999)

knitr::opts_chunk$set(
  echo = TRUE,
  collapse = TRUE,
  comment = "#>",
  error=TRUE
)

Load FWSAkRDRtools into the library and set the working directory (where you would like files to be saved and/or copied from).

library(FWSAkRDRtools)

wd<- "C:/Users/hvincelette/OneDrive - DOI/Documents/Data_management/Test"
